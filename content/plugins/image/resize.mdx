# image resize

Resize images to specified dimensions with various scaling modes.

## Usage

```bash
mediaproc image resize <input> [options]
```

## Description

The `resize` command allows you to change the dimensions of images while maintaining quality. It supports multiple resize modes including fit, fill, cover, and exact dimensions.

## Options

### Dimensions

| Option | Alias | Type | Description |
|--------|-------|------|-------------|
| `--width` | `-w` | number | Target width in pixels |
| `--height` | `-h` | number | Target height in pixels |
| `--size` | `-s` | string | Combined size (e.g., "1920x1080") |

### Resize Mode

| Option | Type | Description |
|--------|------|-------------|
| `--fit` | string | Fit mode: `cover`, `contain`, `fill`, `inside`, `outside` |
| `--position` | string | Position when using cover/contain: `center`, `top`, `bottom`, `left`, `right` |
| `--background` | string | Background color for letterboxing (hex or named color) |

### Quality & Output

| Option | Alias | Type | Default | Description |
|--------|-------|------|---------|-------------|
| `--quality` | `-q` | number | 80 | Output quality (1-100) |
| `--output` | `-o` | string | - | Output file or directory |
| `--format` | `-f` | string | - | Output format (jpg, png, webp, avif) |

### Advanced

| Option | Type | Description |
|--------|------|-------------|
| `--kernel` | string | Interpolation kernel: `nearest`, `cubic`, `lanczos2`, `lanczos3` |
| `--withoutEnlargement` | boolean | Prevent upscaling smaller images |
| `--fastShrinkOnLoad` | boolean | Use faster shrinking during load |

## Examples

### Basic Resize

Resize to specific width (maintains aspect ratio):

```bash
mediaproc image resize photo.jpg --width 1920 -o resized.jpg
```

Resize to specific height:

```bash
mediaproc image resize photo.jpg --height 1080 -o resized.jpg
```

### Exact Dimensions

Resize to exact dimensions (may distort):

```bash
mediaproc image resize photo.jpg --width 1920 --height 1080 -o resized.jpg
```

### Using Size Shorthand

```bash
mediaproc image resize photo.jpg --size 1920x1080 -o resized.jpg
```

### Fit Modes

**Cover** - Crop to fill dimensions:

```bash
mediaproc image resize photo.jpg --size 1920x1080 --fit cover -o cover.jpg
```

**Contain** - Fit within dimensions with letterboxing:

```bash
mediaproc image resize photo.jpg --size 1920x1080 --fit contain --background "#ffffff" -o contain.jpg
```

**Fill** - Stretch to exact dimensions:

```bash
mediaproc image resize photo.jpg --size 1920x1080 --fit fill -o fill.jpg
```

**Inside** - Fit within dimensions (no upscaling):

```bash
mediaproc image resize photo.jpg --size 1920x1080 --fit inside -o inside.jpg
```

### Batch Resize

Resize multiple images:

```bash
mediaproc image resize *.jpg --width 1920 -o resized/
```

Resize all images in a directory:

```bash
mediaproc image resize photos/ --width 1920 -o resized/
```

### With Format Conversion

Resize and convert to WebP:

```bash
mediaproc image resize photo.jpg --width 1920 --format webp -o photo.webp
```

### High Quality Resize

Use Lanczos3 for best quality:

```bash
mediaproc image resize photo.jpg --width 3840 --kernel lanczos3 --quality 95 -o large.jpg
```

### Prevent Upscaling

Only resize if image is larger:

```bash
mediaproc image resize photo.jpg --width 1920 --withoutEnlargement -o resized.jpg
```

## Fit Modes Explained

### cover
- Scales image to cover entire area
- Crops excess parts
- No letterboxing
- Best for: Thumbnails, profile pictures

### contain
- Scales image to fit within dimensions
- Adds letterboxing if needed
- Entire image visible
- Best for: Product images, art

### fill
- Stretches image to exact dimensions
- May distort image
- No cropping or letterboxing
- Best for: When exact size matters

### inside
- Like contain but never upscales
- Best for: Responsive images

### outside  
- Like cover but never downscales
- Best for: Print materials

## Position Options

When using `cover` or `contain`, control image position:

- `center` (default)
- `top`
- `bottom`
- `left`
- `right`
- `top-left`
- `top-right`
- `bottom-left`
- `bottom-right`

```bash
mediaproc image resize photo.jpg --size 1920x1080 --fit cover --position top -o cropped-top.jpg
```

## Performance Tips

1. **Use fastShrinkOnLoad** for large images:
   ```bash
   mediaproc image resize large.jpg --width 800 --fastShrinkOnLoad
   ```

2. **Choose appropriate kernel**:
   - `nearest` - Fastest, lowest quality
   - `cubic` - Good balance
   - `lanczos3` - Best quality, slowest

3. **Batch processing** is faster than individual files

## Common Use Cases

### Responsive Images

Generate multiple sizes:

```bash
mediaproc image resize hero.jpg --width 2048 -o hero-2x.jpg
mediaproc image resize hero.jpg --width 1024 -o hero-1x.jpg
mediaproc image resize hero.jpg --width 512 -o hero-mobile.jpg
```

### Social Media

Instagram post (1080x1080):

```bash
mediaproc image resize photo.jpg --size 1080x1080 --fit cover -o instagram.jpg
```

Facebook cover (820x312):

```bash
mediaproc image resize photo.jpg --size 820x312 --fit cover -o facebook-cover.jpg
```

### Email Newsletters

Optimize for email (max 600px width):

```bash
mediaproc image resize banner.jpg --width 600 --quality 80 -o email-banner.jpg
```
