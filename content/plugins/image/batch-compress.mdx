# batch-compress

Batch compress images for web.

## Usage

```bash
mediaproc image batch-compress <input-pattern> [options]
```

## Description

The `batch-compress` command compresses multiple images with optimized settings for web delivery.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-q, --quality` | number | 80 | Compression quality |
| `--preset` | string | balanced | Preset (maximum, balanced, minimum) |
| `-o, --output-dir` | string | ./compressed | Output directory |
| `-f, --format` | string | - | Convert to format |

## Examples

```bash
# Compress all images
mediaproc image batch-compress "*.jpg" -q 75 -o ./compressed

# Maximum compression
mediaproc image batch-compress "photos/*" --preset minimum -o ./tiny

# Compress and convert to WebP
mediaproc image batch-compress "images/*.jpg" -f webp -q 80 -o ./web-ready
```
