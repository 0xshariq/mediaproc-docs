# bandbool

Boolean operations on color bands/channels.

## Usage

```bash
mediaproc image bandbool <input> [options]
```

## Description

The `bandbool` command performs boolean operations on individual color channels within an image.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-o, --output` | string | - | Output file path |
| `--operation` | string | and | Operation (and, or, xor) |
| `--channels` | string | - | Channels to operate on |
| `-f, --format` | string | - | Output format |
| `-q, --quality` | number | 80 | Output quality |

## Examples

```bash
# AND on RGB channels
mediaproc image bandbool photo.jpg --operation and --channels "rgb" -o result.jpg

# XOR red and blue
mediaproc image bandbool image.png --operation xor --channels "rb" -o mixed.png
```
