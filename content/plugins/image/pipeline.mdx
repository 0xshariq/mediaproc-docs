# pipeline

Chain multiple operations in sequence.

## Usage

```bash
mediaproc image pipeline <input> [options]
```

## Description

The `pipeline` command executes multiple image operations in a single, efficient pipeline for complex workflows.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-o, --output` | string | - | Output file path |
| `--operations` | string | - | JSON array of operations |
| `-f, --format` | string | - | Output format |
| `-q, --quality` | number | 80 | Output quality |

## Examples

```bash
# Resize, crop, and sharpen
mediaproc image pipeline input.jpg \
  --operations '[{"resize":{"width":1000}},{"crop":{"width":800,"height":600}},{"sharpen":{"sigma":1}}]' \
  -o output.jpg

# Complex workflow
```
