# batch-resize

Batch resize multiple images.

## Usage

```bash
mediaproc image batch-resize <input-pattern> [options]
```

## Description

The `batch-resize` command efficiently resizes multiple images at once with consistent settings.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-w, --width` | number | - | Target width |
| `-h, --height` | number | - | Target height |
| `--fit` | string | cover | Fit mode |
| `-q, --quality` | number | 80 | Output quality |
| `-o, --output-dir` | string | ./resized | Output directory |
| `-f, --format` | string | - | Output format |

## Examples

```bash
# Resize all JPGs to 800x600
mediaproc image batch-resize "*.jpg" -w 800 -h 600 -o ./thumbnails

# Resize and convert to WebP
mediaproc image batch-resize "photos/*.png" -w 1920 -h 1080 -f webp -o ./optimized

# Create thumbnails
mediaproc image batch-resize "images/*" -w 300 -h 300 --fit cover -o ./thumbs
```
