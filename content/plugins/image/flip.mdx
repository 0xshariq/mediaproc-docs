# flip

Flip an image horizontally (mirror effect).

## Usage

```bash
mediaproc image flip <input> [options]
```

## Description

The `flip` command creates a horizontal mirror image. This is a lossless operation for most formats and useful for correcting orientation or creating mirror effects.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-o, --output` | string | - | Output file path |
| `-f, --format` | string | - | Output format (jpg, png, webp, avif) |
| `-q, --quality` | number | 80 | Output quality (1-100) |

## Examples

### Basic Flip

```bash
# Flip horizontally
mediaproc image flip input.jpg -o flipped.jpg

# Flip and save as PNG
mediaproc image flip photo.jpg -f png -o mirror.png
```

### Batch Flipping

```bash
# Flip all images in directory
for img in *.jpg; do
  mediaproc image flip "$img" -o "flipped-$img"
done
```

### Combined Operations

```bash
# Flip and convert to WebP
mediaproc image flip input.png -f webp -q 85 -o output.webp

# Flip then rotate (requires piping or separate commands)
mediaproc image flip input.jpg -o temp.jpg && \
mediaproc image rotate temp.jpg -a 90 -o final.jpg
```

## Best Practices

1. **Lossless for JPEG**: Flipping doesn't degrade quality
2. **Mirror Selfies**: Correct selfie mirror effect
3. **Design Symmetry**: Create symmetrical designs
4. **Combine Operations**: Use with other transforms for complex effects
