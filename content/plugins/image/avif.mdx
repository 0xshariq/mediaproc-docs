# avif

Convert and optimize for AVIF format.

## Usage

```bash
mediaproc image avif <input> [options]
```

## Description

The `avif` command provides AVIF-specific optimization for maximum compression with excellent quality.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-o, --output` | string | - | Output file path |
| `-q, --quality` | number | 75 | Quality (1-100) |
| `--effort` | number | 4 | Encoding effort (0-9) |
| `--lossless` | boolean | false | Lossless mode |
| `--chromaSubsampling` | string | 4:2:0 | Chroma subsampling |

## Examples

```bash
# Convert to AVIF
mediaproc image avif photo.jpg -q 75 -o photo.avif

# High quality AVIF
mediaproc image avif image.png -q 85 --effort 6 -o high-quality.avif

# Lossless AVIF
mediaproc image avif logo.png --lossless -o logo.avif
```
