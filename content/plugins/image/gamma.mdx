# gamma

Adjust image gamma correction.

## Usage

```bash
mediaproc image gamma <input> [options]
```

## Description

The `gamma` command adjusts gamma correction to brighten or darken images non-linearly. Lower values darken, higher values brighten.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-o, --output` | string | - | Output file path |
| `-g, --gamma` | number | 2.2 | Gamma value (0.1-3.0) |
| `-f, --format` | string | - | Output format |
| `-q, --quality` | number | 80 | Output quality |

## Examples

```bash
# Lighten image
mediaproc image gamma dark.jpg -g 2.8 -o lightened.jpg

# Darken image
mediaproc image gamma bright.jpg -g 1.8 -o darkened.jpg

# Correct gamma for display
mediaproc image gamma photo.jpg -g 2.2 -o corrected.jpg
```
