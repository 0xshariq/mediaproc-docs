# metadata

View and manipulate image metadata (EXIF, IPTC, XMP).

## Usage

```bash
mediaproc image metadata <input> [options]
```

## Description

The `metadata` command reads, writes, and removes metadata from images. Supports EXIF, IPTC, and XMP metadata standards.

## Options

### Read Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--show` | boolean | false | Display all metadata |
| `--json` | boolean | false | Output as JSON |

### Write Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-o, --output` | string | - | Output file path |
| `--set` | string | - | Set metadata (key=value format) |
| `--copyright` | string | - | Set copyright info |
| `--artist` | string | - | Set artist/author name |
| `--description` | string | - | Set image description |

### Remove Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--strip` | boolean | false | Remove all metadata |
| `--strip-exif` | boolean | false | Remove EXIF only |

## Examples

### View Metadata

```bash
# Show all metadata
mediaproc image metadata photo.jpg --show

# JSON output
mediaproc image metadata image.jpg --show --json

# Save metadata to file
mediaproc image metadata photo.jpg --show --json > metadata.json
```

### Set Metadata

```bash
# Set copyright
mediaproc image metadata photo.jpg --copyright "© 2024 John Doe" -o photo-copyrighted.jpg

# Set artist
mediaproc image metadata image.jpg --artist "Jane Smith" -o credited.jpg

# Set description
mediaproc image metadata photo.jpg --description "Sunset at beach" -o described.jpg

# Set multiple fields
mediaproc image metadata photo.jpg \
  --copyright "© 2024 Studio" \
  --artist "Photographer Name" \
  --description "Product shot" \
  -o complete-metadata.jpg
```

### Strip Metadata

```bash
# Remove all metadata
mediaproc image metadata photo.jpg --strip -o clean.jpg

# Remove EXIF only (keep other metadata)
mediaproc image metadata image.jpg --strip-exif -o no-exif.jpg

# Batch strip metadata
for img in *.jpg; do
  mediaproc image metadata "$img" --strip -o "clean-$img"
done
```

### Privacy Protection

```bash
# Remove location data and personal info
mediaproc image metadata photo.jpg --strip-exif -o anonymous.jpg

# Clean all metadata before sharing
mediaproc image metadata private.jpg --strip -o public.jpg
```

## Metadata Types

### EXIF (Exchangeable Image File Format)
- Camera settings (ISO, aperture, shutter speed)
- Date/time taken
- GPS location
- Camera make/model
- Lens information

### IPTC (International Press Telecommunications Council)
- Copyright
- Credit/attribution
- Keywords/tags
- Description/caption
- Contact information

### XMP (Extensible Metadata Platform)
- Adobe-specific metadata
- Extended copyright info
- Custom fields
- Editing history

## Best Practices

1. **Privacy**: Strip metadata before public sharing
2. **Copyright**: Always add copyright to published work
3. **Attribution**: Include artist info for proper credit
4. **Storage**: Metadata adds minimal file size
5. **Backup**: Keep originals with full metadata

## Common Use Cases

- **Copyright Protection**: Add copyright notices
- **Attribution**: Credit photographers/artists
- **Privacy**: Remove GPS and personal data
- **Organization**: Add descriptions and keywords
- **Web Publishing**: Strip metadata for smaller files
- **Professional Work**: Complete metadata for clients
