# flop

Flip an image vertically (upside down).

## Usage

```bash
mediaproc image flop <input> [options]
```

## Description

The `flop` command flips images vertically, creating an upside-down effect. This is a lossless operation useful for corrections or creative effects.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `-o, --output` | string | - | Output file path |
| `-f, --format` | string | - | Output format (jpg, png, webp, avif) |
| `-q, --quality` | number | 80 | Output quality (1-100) |

## Examples

### Basic Flop

```bash
# Flip vertically
mediaproc image flop input.jpg -o flopped.jpg

# Flop and convert format
mediaproc image flop photo.png -f webp -o upside-down.webp
```

### Batch Processing

```bash
# Flop all images
for img in *.jpg; do
  mediaproc image flop "$img" -o "flopped-$img"
done
```

### Create Reflections

```bash
# Combine with other operations for water reflection effect
# (would require compositing/joining - see composite command)
mediaproc image flop landscape.jpg -o reflection.jpg
```
