# Universal Commands

Universal commands work across the entire MediaProc CLI, regardless of which plugins you have installed. These commands manage plugins and provide essential utilities.

## Command Overview

| Command    | Description                         |
| ---------- | ----------------------------------- |
| `add`      | Install one or more plugins         |
| `remove`   | Remove installed plugins            |
| `list`     | List installed or available plugins |
| `run`      | Run plugin commands                 |
| `validate` | Validate plugin installations       |
| `help`     | Display help information            |
| `create`   | Create a new plugin from templates  |

## add

Install one or more MediaProc plugins.

### Usage

```bash
mediaproc add <plugin-names...>
```

### Options

| Option      | Description                     | Default |
| ----------- | ------------------------------- | ------- |
| `--verbose` | Show detailed installation logs | `false` |

### Examples

**Install single plugin:**

```bash
mediaproc add image
```

**Install multiple plugins:** (Future Plan)

```bash
mediaproc add image video audio
```

**Install with verbose output:**

```bash
mediaproc add video --verbose
```

## remove

Remove installed plugins from your system.

### Usage

```bash
mediaproc remove <plugin-names...>
```

### Examples

**Remove single plugin:**

```bash
mediaproc remove image
```

**Remove multiple plugins:**

```bash
mediaproc remove image video
```

```bash
mediaproc remove image video
```

## list

List installed plugins or view commands available in a plugin.

### Usage

```bash
mediaproc list [plugin-name]
```

### Examples

**List installed plugins:**

```bash
mediaproc list
```

**List commands in a specific plugin:**

```bash
mediaproc list image
```

## run

Run a plugin command.

### Usage

```bash
mediaproc run <plugin> <command> [args...]
```

### Examples

**Run image resize:**

```bash
mediaproc run image resize input.jpg -w 800
```

**Shorthand (without run):**

```bash
mediaproc image resize input.jpg -w 800
```

## validate

Validate plugin installations and dependencies.

### Usage

```bash
mediaproc validate
```

### Examples

**Validate all plugins:**

```bash
mediaproc validate
```

## help

Display help information for commands and plugins.

### Usage

```bash
mediaproc help [command]
mediaproc <plugin> help [command]
```

### Examples

**General help:**

```bash
mediaproc help
mediaproc --help
```

**Command help:**

```bash
mediaproc help add
mediaproc add --help
```

**Plugin help:**

```bash
mediaproc image help
mediaproc image --help
```

**Plugin command help:**

```bash
mediaproc image resize --help
```

## Exit Codes

MediaProc uses standard exit codes:

| Code | Meaning           |
| ---- | ----------------- |
| `0`  | Success           |
| `1`  | General error     |
| `2`  | Invalid arguments |
| `3`  | Plugin not found  |
| `4`  | File not found    |

## Best Practices

### 1. List Commands First

Check available commands before using a plugin:

```bash
mediaproc list image
```

### 2. Validate After Changes

Run validation after installing or removing plugins:

```bash
mediaproc validate
```

### 3. Use Help Commands

Don't memorize everything - use built-in help:

```bash
mediaproc help
mediaproc image resize --help
```

### 4. Use Shorthand

Skip the `run` command for cleaner syntax:

```bash
# Instead of
mediaproc run image resize input.jpg -w 800

# Use
mediaproc image resize input.jpg -w 800
```
