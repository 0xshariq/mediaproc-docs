# Quick Start

Get up and running with MediaProc in 5 minutes.

## 1. Install MediaProc

<Terminal
  commands={[
    { command: "npm install -g @mediaproc/cli" },
    { output: "✓ Installed @mediaproc/cli", isSuccess: true },
  ]}
/>

## 2. Install Image Plugin

<Terminal
  commands={[
    { command: "mediaproc add image" },
    { output: "✓ Plugin installed successfully", isSuccess: true },
  ]}
/>

## 3. Process Your First Image

### Resize an Image

<Terminal
  commands={[
    {
      command:
        "mediaproc image resize photo.jpg --width 1920 --height 1080 -o resized.jpg",
    },
    {
      output: "✓ Resized to 1920×1080 • Saved as resized.jpg",
      isSuccess: true,
    },
  ]}
/>

### Convert Format

<Terminal
  commands={[
    { command: "mediaproc image convert photo.jpg -o photo.webp" },
    {
      output: "✓ Converted to WebP • 2.4 MB → 1.1 MB (54% smaller)",
      isSuccess: true,
    },
  ]}
/>

### Optimize for Web

<Terminal
  commands={[
    {
      command:
        "mediaproc image optimize photo.jpg --quality 85 -o optimized.jpg",
    },
    {
      output: "✓ Optimized • 3.2 MB → 1.8 MB (44% reduction)",
      isSuccess: true,
    },
  ]}
/>

<Terminal
  commands={[
    {
      command: "mediaproc image batch-resize *.jpg --width 800 -o thumbnails/",
    },
    { output: "✓ Processed 12 images • Saved to thumbnails/", isSuccess: true },
  ]}

## Working with Directories

<Terminal
  commands={[
    {
      command:

    },
    {
      output: "✓ Resized 45 images • Saved to output-folder/",
      isSuccess: true,
    },
  ]}
/>

<Terminal
  commands={[
    {

## Plugin Management

### List Installed Plugins

```bash
mediaproc list
```

### View Available Plugins

```bash
mediaproc plugins
```

### Update Plugins

```bash
# Update specific plugin
mediaproc update image

# Update all plugins
mediaproc update
```

### Delete Plugins

```bash
mediaproc delete image
```

## Get Help

### CLI Help

```bash
mediaproc --help
```

### Plugin Help

```bash
mediaproc image --help
```

### Command Help

```bash
mediaproc image resize --help
```
