# Installation

Get MediaProc up and running in minutes.

## Prerequisites

<Callout type="info" title="System Requirements">

Before installing MediaProc, ensure your system meets these requirements:

- **Node.js**: Version 18.0.0 or higher
- **Package Manager**: npm, pnpm, yarn, or bun
- **Disk Space**: At least 500MB for CLI and plugins

</Callout>

## Troubleshooting

### Permission Errors

If you encounter permission errors, use `sudo` (not recommended) or configure npm to install globally without sudo:

```bash
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

### Plugin Not Found

If a plugin command is not recognized:

1. Check if the plugin is installed: `mediaproc list`
2. Reinstall the plugin: `mediaproc add <plugin-name>`
3. Check your PATH environment variable

### Version Conflicts

If you experience issues after an update:

```bash
# Clear npm cache
npm cache clean --force

# Reinstall MediaProc
npm uninstall -g @mediaproc/cli
npm install -g @mediaproc/cli
```
