# Installation

Get MediaProc up and running in minutes.

## Prerequisites

- **Node.js**: Version 18.0.0 or higher
- **Package Manager**: npm, pnpm, yarn, or bun

## Install MediaProc CLI

Choose your preferred package manager:

### npm

```bash
npm install -g @mediaproc/cli
```

### pnpm (Recommended)

```bash
pnpm add -g @mediaproc/cli
```

### yarn

```bash
yarn global add @mediaproc/cli
```

### bun

```bash
bun add -g @mediaproc/cli
```

## Verify Installation

Check that MediaProc is installed correctly:

```bash
mediaproc --version
```

You should see the version number displayed.

## Install Plugins

MediaProc uses a plugin architecture. Install plugins as needed:

### Install Image Plugin

```bash
mediaproc add image
```

### Install Multiple Plugins

```bash
mediaproc add video audio document
```

### Install All Official Plugins

```bash
mediaproc add image video audio document animation 3d stream ai metadata pipeline
```

## Update MediaProc

Keep your installation up to date:

```bash
# Update CLI
npm update -g @mediaproc/cli

# Update specific plugin
mediaproc update image

# Update all plugins
mediaproc update
```

## Uninstall

To remove MediaProc:

```bash
# Uninstall CLI
npm uninstall -g @mediaproc/cli

# Delete specific plugin
mediaproc delete image

# Or uninstall manually
npm uninstall -g @mediaproc/image
```

## Troubleshooting

### Permission Errors

If you encounter permission errors, use `sudo` (not recommended) or configure npm to install globally without sudo:

```bash
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

### Plugin Not Found

If a plugin command is not recognized:

1. Check if the plugin is installed: `mediaproc list`
2. Reinstall the plugin: `mediaproc add <plugin-name>`
3. Check your PATH environment variable

### Version Conflicts

If you experience issues after an update:

```bash
# Clear npm cache
npm cache clean --force

# Reinstall MediaProc
npm uninstall -g @mediaproc/cli
npm install -g @mediaproc/cli
```
